About ASTRO Vending Machine

Welcome to ASTRO

ASTRO is a modern, cyber-themed vending machine system designed to provide a seamless purchasing experience with advanced features and real-time inventory management.

Features

  • Real-time inventory tracking
  • Secure payment processing
  • Admin dashboard for management
  • Low stock alerts
  • Sales analytics and reporting

Technology

Built with Django, PostgreSQL, and modern web technologies, ASTRO provides a robust and scalable solution for vending machine operations.
